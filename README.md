## アドバイス1: 最初のうちに作ってみると良いもの

- 単純にサーバからファイルをダウンロードするだけのクライアント。サーバに用意された以下のコマンドを一通り実行できるようにすること。
    - SIZE
    - GET （2つの書式とも）
    - REP

- 他のホストとの間の通信性能の測定
    - スループット、往復遅延、パケットロス率等、必要に応じて。
